    class SessionResource

    This class is how we make the rails ActiveResource talk to our Api. Any models we make that would talk to the Api should inherit from this model. It handles passing your OAuth token, as well as our endpoint conventions, and the media-types.

def headers
  if !self.access_token.nil?
    @headers = {"Authorization" => "Bearer #{self.access_token}"}
  else
    @headers = {}
  end
end
