    class OauthHelper::Oauth

    The OAuth class implements the various methods needed to handle OAuth as per the Oauth2 gem.

    get_client() click to toggle source

    This method actually does the job of initializing the OAuth2 client with all of the configured information in config/config.yml

    # File lib/oauth_helper.rb, line 46
def get_client()
  apiUrl = "#{APP_CONFIG['api_base']}"
  uri = URI.parse(apiUrl)
  apiUrl = "#{uri.scheme}://#{uri.host}:#{uri.port}"
  return OAuth2::Client.new(APP_CONFIG['client_id'], APP_CONFIG['client_secret'], {:site => apiUrl, :token_url => '/api/oauth/token', :authorize_url => '/api/oauth/authorize'})
end
    get_token(code) click to toggle source

    The final step in this process is to get the acutal OAuth token which is then stored in the @token variable.

    # File lib/oauth_helper.rb, line 55
def get_token(code)
  @code = code
  token = get_client().auth_code.get_token(code , {:redirect_uri => APP_CONFIG['redirect_uri']})
  @token = token.token
  @token
end
